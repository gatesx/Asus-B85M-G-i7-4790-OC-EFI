# Asus-B85M-G-i7-4790-OC-EFI
Asus B85M-G+i7-4790基于OpenCore的EFI<br>
此EFI是基于`OpenCore`正式版本制作的，测试中使用 的系统为`MacOS BigSur `，其中声卡、网卡、USB（除3.0未测试）均正常驱动，推荐各位最后自己改一下三码！
